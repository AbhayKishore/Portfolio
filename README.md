```bash
npm install -g vercel
vercel login

```

### **Option 2: Deploy using Vercel Dashboard**

1. Go to **https://vercel.com**
2. Click **New Project**
3. Import your GitHub repository (`portfolio`)
4. Select **Vite** as the framework
5. Click **Deploy**
